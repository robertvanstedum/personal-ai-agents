# Journal
Chronological build record of the personal-ai-agents project.

Sprint notes, build plans, phase specs, things-not-to-forget. Rough is fine — the messiness is part of the record. This is how the project actually evolved, not a polished retrospective.

**Goes to GitHub as-is.** Anyone reading this can see how a serious personal AI project develops over time, including the parts that were uncertain or incomplete.

---

## What belongs here

- Sprint plans (`SPRINT_*.md`)
- Build plans (`BUILD_*.md`)
- Phase plans (`PHASE_*.md`)
- Current task lists and notes-to-self
- Post-mortems and release notes when something ships

## What doesn't belong here

- Formal specs → `docs/specs/`
- Permanent reference docs → `docs/`
- Temporary scratch notes → `_working/` (gitignored)
- Private material → private repo
