# New Ventures — Guild Cabinet

Looks outward. Finds where Robert's capabilities meet market opportunity.
Distinct from Design-Build (what Robert can do) — New Ventures is about
where Robert applies it.

**Role definition:** [GUILD_CHARTER.md §8.4](../GUILD_CHARTER.md#84-new-ventures)

---

## Scope

| Area | Description | Privacy |
|------|-------------|---------|
| Competitive intel | Who else is building personal AI platforms? B2B API tooling? TM Forum space? | Public |
| Opportunity log | New ideas that bubble up via CoS — structured holding area | Private (mini-moi-private) |
| Job search | Active applications, interview pipeline, follow-ups | Private (mini-moi-private) |

## Privacy Boundary

**Public repo (here):**
- `competitive-intel.md` — research on what others are building (shareable)

**Private repo (mini-moi-private/guild/):**
- `intent-register-private.md` — salary targets, companies to avoid, sensitive goals
- `job-search/` — all job search activity (never in public repo)
- `opportunity-log.md` — private opportunity tracking

## Trigger Phrases

- "What roles are out there?" → job-search review + Grok web search
- "Who else is doing this?" → competitive-intel scan
- "Brief me on the market" → Grok research sweep
- "New opportunity: [X]" → CoS logs, New Ventures evaluates

## Maps To

Grok (primary research engine) + Claude.ai
