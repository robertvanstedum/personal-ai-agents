# Chief of Staff — Guild Cabinet

The most important role in the Guild. Not a domain expert — the generalist who holds
the full picture and ensures every session starts with context, not a blank page.

**Role definition:** [GUILD_CHARTER.md §8.1](../GUILD_CHARTER.md#81-chief-of-staff)

---

## Primary Artifacts

| File | Purpose | When to read |
|------|---------|-------------|
| `intent-register.md` | Robert's 90-day goals, priorities, north stars | Every strategic session |
| `domain-health.md` | Cross-domain status snapshot | "What's the state of the platform?" |
| `situation-room.md` | Escalation protocol + incident log | When something breaks |

## Trigger Phrases

- "Brief me" → read intent-register + domain-health
- "What should I focus on?" → intent-register priorities
- "New idea: [X]" → log in intent-register open questions
- "What's the status?" → domain-health
- "We have a problem with [X]" → situation-room escalation

## Maps To

Claude.ai + Grok in strategic mode.
